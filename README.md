🎮 Game Jah Manager 🎮

Projeto em Python para monitorar e registrar tempo de gameplay automaticamente via terminal.

Funcionalidades
- Cadastro de jogos pelo processo do sistema
- Monitoramento automático de tempo jogado
- Encerramento manual ou automático ao fechar o jogo
- Histórico de sessões salvas em JSON
- Interface via terminal (CLI)

Tecnologias
- Python 3.12
- psutil
- JSON
- Git & GitHub
